1. SPA(Single Page Application)이란?
    - 앱스러운 웹(앱) 개발
    - 앱처럼 페이지 단위로 화면에 구성된 스타일 
    - 서버 데이터(RESTFul API) => 호출 => 화면에 출력하는 스타일
    - SPA 핵심은? 
        - 데이터 바인딩과 라우터 

2. SPA 라이브러리 
    - Vue.js        
    - React.js
    - Angular

3. jQuery vs Vue.js
    - jQuery 
        - 웹페이지에 적합
